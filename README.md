# chip-init
bash scripts for initializing my [Pocket CHIP](https://getchip.com/pages/pocketchip).

## Install
```
$ git clone https://github.com/rhodey/chip-init /tmp/chip-init
$ /tmp/chip-init/chip-init.sh
```
